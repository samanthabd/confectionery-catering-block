== Description ==
Block for package info on catering page, with example and custom icon.

Simple setup with inner blocks template.

Also creates a custom block category and places it at the top of the block inserter.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/catering-package` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
